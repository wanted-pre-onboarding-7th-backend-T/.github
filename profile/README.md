## Git Commit Convention

| 태그이름 | 설명                                                          |
| -------- | ------------------------------------------------------------- |
| feat     | 새로운 기능, 특징 추가                                          |
| style    | 스타일링                                                       |
| fix      | 버그 수정                                                      |
| refactor | 리팩토링 , 코드에영향을 주지 않는 수정                           |
| test     | 테스트 코드 수정, 누락된 테스트를 추가할 때, 리팩토링 테스트 추가  |
| chore    | 빌드 업무 수정                                                 |
| init     | 초기 설정                                                      |
| docs     | 문서 설정                                                      |
| update   | 간단한 수정 작업                                                |
| remove   | 클래스, 메서드 등 삭제                                           |
| comment  | 주석 작성                                                       |
